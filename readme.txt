Trabalho de Algoritmos e Estrutura de Dados

Objetivo: desenvolver uma aplicação com o intuito de escolher a melhor rota, definindo a ordem de sequência de uma campanha de inspeção em poços, utilizando embarção RSV.

O problema se assemelha muito ao do Caxeiro Viajante (Salesman Problem) e como base para o trabalho será utilizado o Artigo "Ant Colony system: A Cooperative Learning Approach to the Traveling Salesman Problem".

O Ant Colony System (ACS) é um método heurístico de busca muito utilizado é problemas de Pesquisa Operacional e Otimização.
